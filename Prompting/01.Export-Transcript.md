퍼즐 앱에서 **빈 칸 2개인 4×4 마방진**을 행·열 합(34)으로 빠르게 채우고, 가끔 **틀린 입력을 1~2번 고친 뒤** 다음 문제로 넘어가는 흐름.

## R-G-I-O

| | 내용 |
|---|---|
| **R (Role · 누가)** | 퍼즐 앱에서 4×4 마방진(0으로 표시된 빈 칸 **정확히 2개**)을 푸는 사용자 |
| **G (Goal · 무엇을)** | 두 빈 칸에 들어갈 **올바른 숫자**를 찾아 문제를 끝내고 **다음 문제로 넘어감** |
| **I (Insight · Mom Test에서 확인된 사실)** | ① 행·열·대각선 합이 34라 **빈 칸이 있는 행**부터 시작함 ② 대부분 **직관적으로 ~1분** 안에 풀림 ③ 행·열 중 하나는 0이 **1개뿐**이라 구조상 막히지 않음 ④ 맞추면 **별도 검증 없이** 바로 다음 문제 ⑤ 가끔 **잘못 입력** → 앱이 틀림 알림 → **수정·재입력 1~2회** |
| **O (Outcome · 성공 기준)** | 아래 5항 — Mom Test 증거와 **1:1** |

## 성공 기준 (Mom Test 증거 ↔ 1:1)

| # | Mom Test 증거 | 성공 기준 |
|---|---|---|
| O-1 | 「1분 정도?」 | 주어진 격자에서 **두 빈 칸 정답을 즉시 산출**할 수 있다 (수동 ~1분 수준의 단순 흐름). |
| O-2 | 「빈 칸이 있는 **행** 기준으로 시작」 | **마법상수 = 34**이며, 0이 **1개**인 행 또는 열에서 `34 − (나머지 칸 합)`으로 **한 칸을 먼저 확정**한다. |
| O-3 | 「0 두 개 + 행·열 중 하나는 0이 1개뿐 → 안 풀리는 문제 없음」 | 빈 칸 **정확히 2개**, **1~16 중복 없음**, 모든 **행·열·대각선 합 = 34**를 만족하는 **유일한** 숫자 쌍을 반환한다. |
| O-4 | 「틀렸다고 나옴 → 수정해서 다시 입력 → **1~2번**」 | 후보 값이 마법상수·중복·대각선 조건을 **위반하면 거부**할 수 있고, 최종 두 값은 **전체 검증을 통과**한다. |
| O-5 | 「다음 문제로 넘어감」(풀린 뒤 재확인 없음) | 출력은 **빈 두 칸에 들어갈 숫자만** 제공한다 (추가 확인·설명 단계 없음). |
